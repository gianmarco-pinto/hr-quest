# HR Quest — La Scalata

Gioco di ruolo gestionale sulla carriera nelle Risorse Umane: parti da neolaureato e scali fino a HR Director, un trimestre alla volta.

## Come si gioca
- Costruisci le tue **competenze** (Soft, Hard, Manageriali) spendendo Punti Azione ogni trimestre.
- Supera la **sfida di fine trimestre**, le cui opzioni dipendono dalle competenze che hai.
- Gestisci **stress**, **stile di vita** e **imprevisti economici** senza andare in burnout.
- Cambia azienda (startup → PMI → corporate → multinazionale) per sbloccare i ruoli più alti.
- Entra in **classifica** in base a velocità, competenze, reputazione e risparmi.

## Tecnologia
Singolo file `index.html`, completamente autosufficiente: nessun build, nessuna dipendenza, icone SVG originali. Si apre in qualsiasi browser e si può servire da qualsiasi hosting statico.

## Avvio in locale
Apri `index.html` nel browser, oppure servi la cartella:

```
python3 -m http.server 4321
```

e vai su http://localhost:4321
